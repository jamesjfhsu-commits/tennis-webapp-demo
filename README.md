# My Tennis Season

A responsive clickable UI demo for an amateur tennis web app. The demo persona is Alex Chen, an intermediate player in Taipei building a Grand Slam-like season called "Road to My First Slam."

## Stack

- Next.js App Router
- React
- Tailwind CSS
- shadcn/ui-style local components
- Radix UI primitives
- lucide-react
- Mock data only

## Local Development

```powershell
npm.cmd install
npm.cmd run dev
```

## Production Build

```powershell
npm.cmd run build
```

The project is configured for static export and GitHub Pages deployment through `.github/workflows/deploy-pages.yml`.
